### 문제가 있었던 것

- 나중에 혼돈을 주지 않기 위해 최대한 자세하게 와이어프레임을 짜다 보니 시간이 너무 많이 소요됐다.
- 각자가 생각하는 디테일 차이가 커서 의견을 하나로 취합하기 어려웠다. 그래도 팀원들이 자기의견을 잘 피력하고, 남의 의견을 잘 경청해줘서 수월하게 끝냈다.
- 오늘 와이어프레임을 완성하려고 했는데, 생각보다 결정할 게 많아서 디테일적인 부분은 내일 마저 이야기 해야할 것 같다. 오늘 끝내겠다는 계획 실패....
- 비가 너무 많이 온 탓에 전체적으로 어수선했다.

### 새로 배운 것

- Commit convention의 필요성을 다시금 깨달았다.
- 코치님 강의를 통해 UX/UI를 더 풍성하게 해줄 사이트들을 많이 배웠다. 집 가서 더 찾아보고 어떤 사이트가 가장 적합할지 생각해봐야겠다.
- 프로그램 하나 만드는 게 쉬운 일이 아니라는 것을 뼈저리게 배웠다. 얘기 할수록 해야할 게 늘어나고 규모가 커진다. 디테일이 진짜 중요한 것 같다.

### 아직 잘 모르는 것, 부족한 것

- 이전 프로젝트 때 무서워서 git 협업을 안해본터라 어떻게 해야할 지 아직 막막하다. 팀원들이 큰 흐름은 알려줬는데, 토요일에 정처기 시험 끝나면 관련 강의를 들어야겠다.
- 디테일한 부분을 어떻게 구성해야 할지 아직 못 정했다. 내일 중으로 꼭 완성해야겠다. 화이팅! 

### 잘 한 것

- 와이어프레임을 거의 완성했다. 긴긴 여정이었다.
- 아이디어 회의 때 남들의 의견을 취합, 중간에서 잘 정리해서 회의를 원활하게 잘 끝낸 것 같다.
- 미루고 미루던 git convention, commit convention, branch convention을 작성했다.
